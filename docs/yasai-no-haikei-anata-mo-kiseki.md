# あなたも軌跡 design note

## 位置づけ

`あなたも軌跡` は、`軌跡` の標準機能ではなく、有料追加機能・ギフト向け機能として扱う。

標準機能は、商品・作品に添えたQRから共通ストーリーを見せる体験である。`あなたも軌跡` はその上に、顧客へ商品を手渡す瞬間の手元動画を、その人専用のQR体験として追加する。

## 想定用途

- 焼き菓子
- ギフト
- 工芸品
- 記念品
- その場で渡すこと自体に意味がある小さな商品・作品

農家PoCでは顧客別の手渡し動画は使わず、共通ストーリーのみを使う。

## 基本構成

- 共通ストーリー
- 顧客ごとの手渡しクリップ
- 顧客専用QR
- 1日限定表示
- 当日中のダウンロード導線

顧客専用URLは推測できないランダムURLにする。連番IDや作り手IDから推測できるURLにはしない。

## 接客体験の原則

接客の空気を壊さないことを最優先にする。

- 商品を渡すこと自体も接客であり、空間演出の一部として扱う。
- 撮影都合で不自然な動作を強制しない。
- 5秒以内に急いで渡させるような設計にしない。
- 店員は焦らず、通常の接客として自然に商品を手渡す。
- 顧客同意を必須にする。
- 撮影対象は手元だけにする。
- 無音で録画する。
- 顔や声を入れない運用にする。

## 採用する撮影方式

1. 店員が撮影ボタンを押す。
2. 最大15秒程度、手元だけを無音で録画する。
3. 店員は顧客に自然に商品を手渡す。
4. アプリが手渡し部分を自動推定する。
5. 必要に応じて最小限の手動補正を行う。
6. 共通ストーリーの最後に手渡しクリップを追加する。

自動推定は、まず画像認識による手検出ではなく、中央の受け渡しエリアの動き検出を基本にする。手認識は将来の補助機能として扱う。

## 最小手動補正

自動切り出し後に、以下だけを用意する。

- この範囲で保存
- 前後を広げる
- 撮り直す

細かい動画編集や複雑なタイムライン編集は作らない。

## 今回やらないこと

- 常時撮影
- QRカードの自動画像認識トリガー
- 完全自動手認識
- 顧客アカウント
- 決済
- 高度な動画編集
- 長期保存
- SNS共有
- 顔ぼかしAI

## PoC方針

- 焼き菓子販売で `あなたも軌跡` を試す。
- 農家PoCでは顧客別手渡し動画は使わず、共通ストーリーのみとする。
- 最初は無料または少数限定で反応を見る。
- 反応が良ければ、QRカード単位の有料オプションにする。
- 価格案として +200円を検討するが、最初から固定しない。
